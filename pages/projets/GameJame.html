Optimisation des données pour une Game Jam

Contexte

Ce projet était un projet réalisé dans le cadre de l’université. Pour ce projet, nous avons eu un jour et demi et nous étions 5. Notre projet vise à créer une base de données pour une Game Jam, un événement où des équipes de développement de jeux vidéo s'affrontent pour créer le meilleur jeu dans un laps de temps limité. La base de données doit gérer les participants, les groupes, les salles, les jurys, les sessions, ainsi que les notes attribuées à chaque jeu.

Missions réalisées

  • Création de la structure de base de données
  • Ajout de données représentatives
  • Écriture de requêtes SQL analytiques
  • Test, validation et documentation complète

Déroulement du projet

  1. Modèle conceptuel (MCD)
  2. Implémentation MySQL
  3. Création des tables et insertion des données
  4. Requêtes analytiques, optimisation et vérifications
  5. Documentation complète du processus

Compétences mobilisées

  • Modélisation relationnelle (MCD, MLD)
  • Requêtes SQL avancées
  • Analyse, rigueur, documentation technique

Illustrations

  • Modèle Conceptuel & Logique
  • Exemples de requetes

Conclusion réflexive

La création et l'optimisation d'une base de données pour une Game Jam présentent des défis uniques, mais offrent également des opportunités pour améliorer l'organisation de l'événement et l'expérience des participants. Une gestion efficace des données nous permettra une évaluation précise des performances des équipes, favorisant ainsi l'innovation et la compétitivité. Dans le temps imparti, notre travail était cohérent et répondait aux besoins énoncer.
